## 定义与规律

1905 年爱因斯坦提出**光子说**:光本身就是一份一份不可分割的能量子,每一份叫**光子**,频率为 $\nu$ 的光,其光子能量 $\varepsilon=h\nu$。

光电效应的图景:**一个电子一次只吸收一个光子**,获得能量 $h\nu$;逸出金属表面至少要克服束缚做功 $W_0$(**逸出功**,只由金属材料决定)。**表面电子逸出耗能最少**,剩余动能最大,由此得**光电效应方程** $E_k=h\nu-W_0$(人教版符号:$E_k$ 特指光电子的**最大初动能**)。

光子说逐条解释了三条实验规律:频率决定单个光子能量→存在截止频率、$E_k$ 只随 $\nu$ 增大;光强决定单位时间到达的光子数→饱和电流随光强增大;吸收一次完成→瞬时性。**密立根**用约十年的实验验证了光电效应方程并测出普朗克常量 $h$,爱因斯坦因光电效应理论获 1921 年诺贝尔物理学奖。

**四组必须分清的概念(学案高频辨析):**

1. **初动能 vs 最大初动能**:金属内不同深度的电子逸出时耗能不同,逸出后动能从 0 连续分布到某个上限;方程里的 $E_k$ 是这个**上限**(最大初动能),对应表面、耗能最少的电子。凡题目说"光电子的初动能等于 $h\nu-W_0$"一律错——只有"最大初动能"才等于。
2. **光电流 vs 饱和光电流**:加正向电压时电流随电压升高而增大;当所有逸出的光电子都被收集,电流不再增大,达到**饱和光电流**。饱和光电流只由**单位时间逸出的光电子数**决定,即只由光强(光子数)决定,与频率无关。
3. **光子要"恰好",粒子只需"够"**:**光子与电子作用是"整份吸收、要恰好凑上"的**——能量高于逸出功的部分变成动能,不足则完全不逸出,不能积攒两个光子;而用**加速电子/粒子轰击**原子使其电离或跃迁时,粒子只需能量**大于等于**所需值即可(多余的仍留在粒子上),这是能级跃迁题里"光子照射"与"电子碰撞"的关键分水岭。
4. **遏止电压 vs 加速电压**:遏止(截止)电压 $U_c$ 是使光电流恰好减为零的**反向**电压,反映的是**最大初动能**;不要与把电子加速的正向电压混为一谈。

## 核心公式

$$\varepsilon=h\nu=\dfrac{hc}{\lambda}$$
光子能量;$h=6.63\times10^{-34}\ \mathrm{J\cdot s}$,$c=\lambda\nu$ 适用于真空中的光。

**考点核心——三个重要关系式**(高考解答题的三条腿,务必成组记忆):

$$E_k=h\nu-W_0$$
**能量来源式**(光电效应方程);$E_k$ 是光电子的**最大**初动能。仅当 $\nu\ge\nu_c$ 时发生光电效应;$\nu=\nu_c$ 时电子恰能逸出,$E_k=0$。

$$W_0=h\nu_c$$
**材料判据式**($\nu_c$ 即截止/极限频率,也常记作 $\nu_0$)。逸出功与截止频率互相换算,两者都**只由金属决定**;判断某金属能否发生光电效应,就是比较 $h\nu$ 与 $W_0$(等价于比较 $\nu$ 与 $\nu_c$)。

$$E_k=eU_c=\dfrac{1}{2}mv_{\max}^2$$
**测量桥梁式**(遏止电压):具有最大初动能的光电子恰好被反向电场"刹停",由动能定理得出($m$ 为电子质量)。这一式把"看不见的最大初动能"翻译成"可测的遏止电压",是实验取数的入口。

三式联立可得线性关系 $eU_c=h\nu-W_0$,即 $U_c=\dfrac{h}{e}\nu-\dfrac{W_0}{e}$。

**图像的斜率与截距(高频考法):**
- $U_c\text{-}\nu$ 图:斜率 $=\dfrac{h}{e}$(对**所有金属都相同**,是测普朗克常量 $h$ 的原理),横轴截距 $=\nu_c$(截止频率),纵截距的绝对值 $=\dfrac{W_0}{e}$。
- $E_k\text{-}\nu$ 图:斜率 $=h$,横轴截距 $=\nu_c$,纵轴负截距 $=-W_0$。
- 换不同金属:直线**平行上下平移**(斜率不变,截止频率/逸出功不同)。

换算与估算:$1\ \mathrm{eV}=1.60\times10^{-19}\ \mathrm{J}$;$hc\approx1240\ \mathrm{eV\cdot nm}$(教学估算工具,**非教材公式**,解答题中需从 $h$、$c$ 算起)。

## 物理直觉

把方程读成一笔账:光子送来一笔收入 $h\nu$,电子出金属的"门"要交一笔由金属定价的固定"赎身费"$W_0$,口袋里**最多**剩下 $h\nu-W_0$。为什么是"最多"?住得深的电子在路上还要额外消耗,到门口时剩得更少——所以光电子的初动能有一个从 0 到 $E_k$ 的分布,方程只给上限。

再把三个关系式想成一条流水线:$\nu$(光的属性)先经 $E_k=h\nu-W_0$ 换成最大初动能,再经 $E_k=eU_c$ 换成能测的电压。$W_0=h\nu_c$ 则是这台机器的"起动门槛":$\nu$ 没过 $\nu_c$,机器根本不转。

## 最小例子

钠的逸出功约 $2.3\ \mathrm{eV}$,用波长 $400\ \mathrm{nm}$ 的紫光照射:光子能量 $\varepsilon=\dfrac{hc}{\lambda}\approx\dfrac{1240}{400}\ \mathrm{eV}\approx3.1\ \mathrm{eV}$,故 $E_k=3.1-2.3=0.8\ \mathrm{eV}$,遏止电压 $U_c=0.8\ \mathrm{V}$。注意 $U_c$ 的伏特数与 $E_k$ 的电子伏数在数值上恰好相等——这是 $eU_c=E_k$ 的直接结果。

**学案真题型例子(体会"用哪个式子"):**
- **给遏止电压反求逸出功**:某单色光照射金属,测得光电子最大初动能 $E_k=2.75\ \mathrm{eV}$(即 $U_c=2.75\ \mathrm{V}$),已知入射光子能量 $h\nu$,则逸出功 $W_0=h\nu-E_k$。这类题的套路永远是"先定 $h\nu$,再用方程解出未知的那一个"。
- **比较两种金属能否逸出**:铝的逸出功比钙大(截止频率更高)。同一束光照射,可能"钙有光电子、铝没有";要判断就把 $h\nu$ 分别与两者的 $W_0$ 比较——过了 $\nu_c$ 才算数。这也说明**截止频率越高的金属越不容易发生光电效应**。

## 常见误区

1. **$h\nu-W_0$ 只是"最大"初动能**:光电子初动能有从 0 到 $E_k$ 的分布,不能说"所有光电子动能都等于 $h\nu-W_0$";只有从表面逸出、耗能最少的电子才达到 $E_k$。
2. **逸出功 $W_0$ 只由金属决定,与入射光无关**:换更高频率的光,$W_0$ 不变,变的是 $E_k$。
3. **一个电子一次只吸收一个光子**(高中框架):不能"攒两个红光光子凑够能量"。这正是**频率低于截止频率的光,无论光强多大、照射多久,都照不出光电子**的原因。
4. **"频率不变时,增大光强可增大 $E_k$ 或 $U_c$"是错的**:光强只增加单位时间到达的光子数,不改变单个光子的能量,所以 $E_k$、$U_c$、截止频率都不变,增大的只是单位时间逸出的光电子数(饱和电流)。
5. **单位混乱**:$E_k$ 用 $\mathrm{eV}$ 时,$U_c$ 数值相同但单位是 $\mathrm{V}$;代入 $\dfrac{1}{2}mv^2$ 求速度前必须把 $E_k$ 换成焦耳。
6. **密立根验证光电效应方程并测出 $h$**——只需记这个结论;其装置与数据处理超纲,且**勿与密立根油滴实验(测元电荷 $e$)混淆**,两者是不同实验。
7. **图像斜率认错**:$U_c\text{-}\nu$ 图斜率是 $\dfrac{h}{e}$ 而非 $h$;$E_k\text{-}\nu$ 图斜率才是 $h$。看清纵轴是"电压"还是"动能"再取斜率。
8. **把"能量够"当成"一定逸出"**:光照射时必须 $h\nu\ge W_0$(单个光子"要恰好凑上");而电子碰撞使原子电离/跃迁时只需能量 $\ge$ 阈值即可。两种作用机制的判据不同,不能混用。
9. **学案判断题辨析要点**:①"光电子的最大初动能与入射光强度成正比"——错,只与频率有关。②"只要光足够强就能发生光电效应"——错,取决于频率是否达到截止频率。③"入射光频率增大,饱和电流一定增大"——错,饱和电流由光强(光子数)决定,与频率无直接关系。④"逸出功就是光电子的最大初动能"——错,逸出功是逸出所需的最小功,与初动能是两回事。

## 超纲边界

- 密立根验证光电效应方程实验的装置细节与数据处理不考。
- 光子静质量为零、相对论能量-动量关系的推导不考。
- 多光子吸收、逸出功(功函数)的固体物理解释不展开。
- 康普顿效应只作**定性**了解(光子与电子碰撞遵守能量、动量守恒,散射光波长变长),其定量公式(康普顿波长、散射角关系)不考;可用来佐证"光子有动量 $p=\dfrac{h}{\lambda}$"。
- 维恩位移定律、玻尔模型的定量推导、里德伯公式的定量应用、不确定性关系的定量计算,均**不考**。
- $hc\approx1240\ \mathrm{eV\cdot nm}$ 只作选择题心算工具,解答题必须用 $h$、$c$ 原始数据列式。

## 关联题型

- 光电效应方程计算($\lambda\to\varepsilon\to E_k\to U_c$ 链条计算),或反向:给 $U_c$、$\nu$ 求 $W_0$/$\nu_c$。
- **三关系式联立**:已知逸出功与入射频率求遏止电压,或已知两组 $(\nu,U_c)$ 数据反求 $h$ 与 $W_0$(等效于求直线斜率与截距)。
- **$U_c\text{-}\nu$ / $E_k\text{-}\nu$ 图像题**:读斜率($\dfrac{h}{e}$ 或 $h$)、横截距($\nu_c$)、纵截距($\dfrac{W_0}{e}$ 或 $W_0$);多金属平行线的比较(如"2024 黑吉辽卷"级别的图像辨析考法)。
- 用光子说解释实验规律的简答/辨析、学案判断题型。
- 与能级跃迁联动:氢原子跃迁发出的光照射金属能否发生光电效应(跨单元综合,需用 $E_1=-13.6\ \mathrm{eV}$、$E_n=\dfrac{E_1}{n^2}$ 能级数据算出光子能量再与 $W_0$ 比较);注意区分"光照射(要恰好)"与"电子碰撞(只需够)"。
- 与康普顿效应/光的动量联动:由 $\varepsilon=h\nu$ 和 $p=\dfrac{h}{\lambda}$ 定性判断光子既有能量又有动量(定量超纲)。

## 讲法要点

1. **讲授节拍**:先回放上一考点的两个"经典解释不了":为什么有截止频率?为什么瞬时?→ 一句话抛出光子说("光的能量本来就是一粒一粒的")→ 用记账类比现场写出方程,逐项指认收入/支出/结余 → **让学生自己**用光子说逐条"回收"三条实验规律 → 算一遍钠的最小例子。
2. **把三关系式当"一套工具"教**:$E_k=h\nu-W_0$(来源)、$W_0=h\nu_c$(门槛)、$E_k=eU_c$(测量),画一张"$\nu\to E_k\to U_c$"流水线图,让学生看到解答题其实就是在这条线上"给两头求中间"。
3. **学生最容易懵的点是"最大"二字**:必须用"住得深的电子路上还要耗能"把动能分布立住,否则遇到"所有光电子动能相同"的选项就会错。
4. **第二易懵点是 $eU_c=E_k$ 从哪来**:现场用动能定理推一遍(最快的电子恰好被刹停),不要当结论硬塞。
5. **图像专项**:同一坐标里画钠、钙、铝三条平行线,现场标斜率、截止频率、纵截距,反复强调"斜率对所有金属相同=测 $h$ 的原理"。
6. **点透两种作用机制**:光子"要恰好凑上"、粒子"够了就行",用一道能级题当场对比,防止跨单元综合失分。
7. **检查性提问**:①"方程里哪个量由光决定、哪个由金属决定?"②"为什么说 $h\nu-W_0$ 是'最大'初动能?"③"$E_k=0.8\ \mathrm{eV}$,遏止电压是多少?"④"频率不变、光强加倍,$E_k$、$U_c$、饱和电流各怎么变?"⑤"$U_c\text{-}\nu$ 图斜率代表什么?换金属后线怎么动?"⑥"光照射要 $h\nu\ge W_0$,那电子碰撞使原子跃迁需要什么条件?"⑦"密立根这个实验测的是 $h$ 还是 $e$?"
8. **收尾**:让学生自己填"实验规律 ↔ 光子说解释"三行对应表(截止频率↔单光子能量门槛、饱和电流↔光子数、瞬时性↔一次吸收),再补一张"三关系式↔各自用途"小表,填完即完成从现象到理论、再到解题工具的闭环。
