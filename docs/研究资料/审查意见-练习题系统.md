OVERALL:
这份设计的骨干方向明显高于平均水平:分层降级、『能代码判绝不 AI 判』、标准答案注入、参数化图形、变式重练等核心决策都站得住,『诚实风险清单』的姿态也正确。但它的『诚实』大量建立在无出处的伪精确数字和失实/张冠李戴的引证上(88% 转写率、60-80% 可用率、FeynmanBench、菁优案『确认』、『新课标I卷』物理),对一位无法自行核验的创始人,这种『看起来严谨』比明说不知道更危险。方案内部还有多处自相矛盾(变式管线的 V0/V1 归属、原则5与即兴变式、错因码与考点树的断层),说明闭环没有真正推演过一遍。对『一人+Claude Code』的配置,电路渲染器、录题工时、教研门槛被系统性低估,而实验题缺失与中国市场合规(备案、未成年人数据出境、Claude 大陆可用性)是两个决定产品能否接触真实学生的整块遗漏。最薄弱的一环是批改可信度的兜底机制——它把最后防线压在 LLM 自报 confidence 这个当前模型给不出的能力上,上线前必须换成双模型或多次采样一致性等外部信号。

FACTUAL_ERRORS:
- 「2024·新课标I卷 T14」作为物理题来源标签是错的:不存在「新课标I卷物理」。新课标I/II卷仅指教育部教育考试院命制的语文/数学/外语统考卷;新高考模式下物理由各省(市)自主命题([2024年各省高考模式解析](https://news.qq.com/rain/a/20240628A00L8800))。2024年只有尚未改革的老高考省份(河南、云南、山西、西藏、新疆)的理综被称作「新课标卷」,且到2026-2027届这些省份已全部转入新高考、理综消亡。整份方案「对齐高考实际」的叙事建立在「物理有一张全国卷」的过时想象上。
- 「菁优诉豆丁案确认解析是自建版权资产」失实:该案(深圳菁优诉豆丁世纪,(2019)粤03民再171号)恰恰证明解析的版权保护高度不确定——一审认定构成作品,二审认定常规演算不构成作品,再审虽认可有独创性点评的解析可保护,但最终驳回了菁优全部诉讼请求([案件评析](http://ipr.mofcom.gov.cn/article/gnxw/sf/zz/zzbq/202008/1953600.html))。正确结论是「含个性化点评的解析可能受保护,常规解题步骤未必」,把它写成『确认』会让创始人对资产价值产生错误预期。
- 「真题截图内测阶段合法可用」「内测使用无版权风险」表述过强且前后矛盾(4.1说『风险极低』,4.4说『无版权风险』):个人学习研究可主张合理使用、风险确实低,但『合法』『无风险』是法律结论级断言,不该出现在给非法律背景创始人的文档里。另外省级命题时代,真题权利主体分散在各省教育考试院,将来商用授权对象因省而异,方案未提。
- FeynmanBench 引用张冠李戴:该基准([arXiv 2604.03893](https://arxiv.org/abs/2604.03893))评测的是多模态模型对粒子物理费曼图的推理理解(拓扑、守恒律、散射振幅),与「AI 生成高中物理示意图 SVG 的质量」完全不是一回事,不能作为1.2.1结论的证据。同表中 60-80%、50-70% 等『可用率』是无出处的伪精确数字(虽标注『估』,但会被当成规划依据)。
- 手写转写「88% 可接受率(GPT-4.1-mini 配专用提示词,Claude 同量级)」无出处:真实转写率随照片质量、涂改、图文混排剧烈波动;高三物理解答(含受力图、划线删改、多栏书写、图上标注)比任何『手写数学作业』基准都难。88% 不能当规划常量,「Claude 同量级」纯属假设。
- 「符号表达式等价判断程序上是 CAS 难题」是错误的技术论断:一般等价判定理论上不可判定,但高中物理表达式(多项式/三角/根式)完全在 SymPy、mathjs 等现成 CAS 的可靠工作范围内(simplify(a-b)==0 类判定成熟)。方案却退化到『字符串白名单归一化+让 LLM 判等价』,把最该用确定性工具的地方交给了模型,与自己的设计原则1相悖。
- 以 LLM 自报 confidence(如 0.88)作为措辞分级兜底不成立:大模型自评置信度校准性差、系统性偏高,「confidence<0.7 降级措辞」建立在模型给不出的能力上。需要的是外部信号:双模型独立批改一致性、同模型多次采样一致性——方案把双模型交叉放在 V1,恰恰把唯一可靠的置信来源推迟了。
- 「连带分:前步错后步方法对只扣前步的分——这是高考评分的真实规则」过度简化:真实高考评分细则按方程/得分点给分,前步错误后,后续的『方程/方法分』通常可得,但『结果分』通常不给,并非『只扣前步』。照此写死会与真实评分标准系统性偏差。
- 4.3 sanity 规则表本身有物理错误:「v<340 m/s、a<50 m/s²」作为全局数值域检查,会直接误杀天体运动(第一宇宙速度 7.9 km/s)、带电粒子在电磁场中运动(v~10^7 m/s,a 可达 10^13-10^14 m/s²)等高考核心题型。数值域必须按章节/情境分表,不能全局一刀切。
- 多条『调研教训』属无法核实的引证却被当成工程结论依据:学而思批改被投诉、Khanmigo 算错 343-17、讯飞 97% 一致性与 4000+ 错因标签、Khan Academy 官方博客『验证喂标准答案』。方向或许不错,但对一位无法自行验证的创始人,所有引证应标注可信度或给出出处,而非以既成事实口吻陈述。
- 方案通篇未提已废止的考试大纲(这点正确,2019年底已取消),但也没有锚定现行《普通高中物理课程标准(2017年版2020年修订)》和各省教材版本(人教/粤教/鲁科/教科)差异——『考点树』将对齐什么标准、覆盖哪些选择性必修内容,全文没有交代。
