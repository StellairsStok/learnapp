# 研究资料使用说明

本目录是 2026-07-02 多代理设计流程的原始产出,共三类文件:

- **调研-*.md**:联网调研的事实摘要(课标制度、竞品、技术实践、版权),绝大多数条目带来源链接,少数标注"未经检索验证"。
- **设计稿-*.md**:五个维度的详细设计初稿。**内含已知错误,不能单独作为实施依据**。
- **审查意见-*.md**:对应设计稿的批判性审查(事实核查 / 必须修正 / 建议优化)。

**阅读规则:任何一份设计稿必须与同名审查意见一起读。** 实施某个模块时,以《00-项目计划书》的裁决为准——计划书已把审查意见中的修正融合进去;设计稿仅作为工程细节(数据结构、流程、示例)的参考库。

已确认的设计稿主要错误(计划书中已修正),包括但不限于:

1. "大题写超纲方法不得分"是错误论断(通行阅卷做法是数学正确的解法参照给分,真实风险是步骤分)。
2. "2024·新课标I卷 T14"这类物理题来源标签不存在——新高考物理由各省自主命题,没有全国统一物理卷。
3. 各处"85-93%""88% 转写率""95%+ 合规"等数字均为未经实测的估计值,不得当作事实引用。
4. LLM 自报的 confidence 不是可靠的置信度,不能作为安全机制(要用多次独立采样一致性等外部信号)。
5. 表达式等价判定应使用现成 CAS(SymPy/mathjs),不是"CAS 难题"。
6. 学业质量分级:2020 修订版为 5 级(水平 4 = 选择性考试依据);2025 修订版据全文转载改为 3 级(选择考不超过水平 3)——以官方 PDF 终核为准,终核前卡片不写死该字段。
7. "菁优诉豆丁案确认解析是版权资产"表述过强:含个性化点评的解析可能受保护,常规解题步骤未必。
8. 自测阶段 Claude API 的大陆可用性不是"零障碍",见计划书风险章。
